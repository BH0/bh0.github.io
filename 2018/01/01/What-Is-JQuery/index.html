What Is JQuery

2018-01-01

Please note that this is not a JQuery tutorial but aims to clear up some of the misconceptions regarding JQuery.

When you search �what do I need to know to be a web developer?� one of the most common requirements is JQuery, and while this is somewhat true - it is not entirely accurate. JQuery is a collection of Javascript code, some call it a framework, I call it a library and others seem to think its an alternative to Javascript. JQuery is Javascript, the JQuery files all contain Javascript and JQuery do the same thing.

Well, if they are the same - why is JQuery so popular? Simple answer: laziness (or convenience if you prefer). There are three main purposes for JQuery: DOM selection + manipulation, AJAX requests, and lastly, interactivity effects such as �fade out when clicked, �, other uses for JQuery include styling elements easier, JQuery�s utilities and event handling.

JQuery is also used for AJAX requests - getting data from an external source.Whats great about JQuery�s AJAX code is that the various different types of requests (get, post etcetera) can be specified within the code much easier and you can define the behaviour of the request much quicker.

JQuery also offers a lot of utilities which are commands which aid the develoepr such as �jquery.each();�.

There is also something called JQuery UI. From the website: �JQuery UI is a curated set of user interface interactions, effects, widgets and themes built on top of the JQuery Javascript library�.

And lastly, JQuery Mobile - which is categorized as a framework - a language which makes mobile application development possible using web technology.

There are also many JQuery plugins which is essentially code not large enough to be defined as a library that extend JQuery made by the public.

Some facts about JQuery: JQuery is often cached - meaning your browser stores a local copy of the library so that it doesn’t have to re-acquire JQuery from elsewhere each time you visit the website. Thus improving load speeds. Content management systems such as WordPress comes with JQuery already and it is best to use WordPress�s JQuery. JQuery is open source so you can improve it via their Github repository.

So, remember, when you use JQuery - you’re still using Javascript. But never the less - learn vanilla Javascript, not JQuery - when you really need JQuery - that’s when it helps to learn JQuery.

As Javascript can do everything JQuery can do, many developers recommend not using Jquery unless your application truly will make extensive use of Jquery, but if not then it is recommended to just use Javascript or a library smaller than Jquery designed to accomplish the specific task you want to accomplish, such as Axios for AJAX requests. This means you don’t have to have a massive library like Jquery bloating up your application when you’re only using a small percentage of JQuery.
